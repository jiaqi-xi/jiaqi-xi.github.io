Convex Optimization(2): Convex Sets
发表于 2021-04-07| 更新于 2021-04-07|Course Notes
阅读量:

I am taking Convex Optimization this semester (2021 Spring). The notes are based on my understanding of the course materials provided by Prof. Zhouchen Lin. The notes only offer concise definition and formulas with few proofs and examples.

In this chapter, we introduce convexity. We define convex sets first and will discuss convex functions in the next chapter. We go from affine sets to convex sets and cones. Later on, we will talk about some operations that could preserve convexity. Also, we will discuss about how to separate two convex sets.

Affine and convex sets#

Affine#

  • Line segment

    The line passing through two given points $ \mathbf{x}{1}, \mathbf{x}{2}$ can be described parametrically by θ\thetaθ:

    y=θx1+(1−θ)x2=x2+θ(x1−x2)\mathbf{y}=\theta \mathbf{x}_{1}+(1-\theta) \mathbf{x}_{2} = \mathbf{x}_{2}+\theta( \mathbf{x}_{1}- \mathbf{x}_{2}) y=θx1​+(1−θ)x2​=x2​+θ(x1​−x2​)

  • Affine sets

    A set is affine if for any $ \mathbf{x}{1}, \mathbf{x}{2}\in {C}, \theta\in\mathbb{R}$, we have θx1+(1−θ)x2∈C\theta \mathbf{x}_{1}+(1-\theta) \mathbf{x}_{2}\in{C}θx1​+(1−θ)x2​∈C.

    e.g. The solution set of a linear equation is an affine set:

    Ax1=b,Ax2=b⇒A(θx1+(1−θ)x2)=b\mathbf{Ax}_1=\mathbf{b}, \mathbf{Ax}_2=\mathbf{b}\Rightarrow \mathbf{A}(\theta \mathbf{x}_{1}+(1-\theta) \mathbf{x}_{2})=\mathbf{b} Ax1​=b,Ax2​=b⇒A(θx1​+(1−θ)x2​)=b

    In reverse, every affine set can be expressed as the solution set of a system of linear equations.

  • Affine hull

    Given a set C\mathcal{C}C, generalize to finite(to ensure the affine combination to be finite) multiple points and we have its affine hull:

    aff⁡C={θ1x1+…+θkxk∣x1,…,xk∈C,θ1+…+θk=1}\operatorname{aff} C=\left\{\theta_{1} \mathbf{x}_{1}+\ldots+\theta_{k} \mathbf{x}_{k} \mid \mathbf{x}_{1}, \ldots, \mathbf{x}_{k} \in C, \theta_{1}+\ldots+\theta_{k}=1\right\} affC={θ1​x1​+…+θk​xk​∣x1​,…,xk​∈C,θ1​+…+θk​=1}

    We can define relative interior and relative boundary of an affine set. By relative, we mean that when dimC<dimB(x,r)\text{dim} {C} < \text{dim} {B}(\mathbf{x}, r)dimC<dimB(x,r), only B(x,r)∩aff⁡C⊆CB(\mathbf{x}, r) \cap \operatorname{aff} {C} \subseteq {C}B(x,r)∩affC⊆C is required (not the whole ball).

Convex#

  • Convex sets

    A set is convex if for any x1,x2∈C,0≤θ≤1\mathbf{x}_{1}, \mathbf{x}_{2}\in {C}, 0\le\theta\le 1x1​,x2​∈C,0≤θ≤1, we have θx1+(1−θ)x2∈C\theta \mathbf{x}_{1}+(1-\theta) \mathbf{x}_{2}\in{C}θx1​+(1−θ)x2​∈C. (i.e. line segment between any 2 points lies in the set)

    • Not necessary to be closed
    • ∅\empty∅ is also convex

Cone#

A set CCC is a cone if for every x∈C,θ>0\mathbf{x}\in C,\theta>0x∈C,θ>0, we have θx∈C\theta\mathbf{x}\in Cθx∈C.

A set CCC is a convex cone if for any x1,x2∈C,θ1,θ2>0\mathbf{x}_1,\mathbf{x}_2\in C, \theta_1,\theta_2>0x1​,x2​∈C,θ1​,θ2​>0, we have θ1x1+θ2x2∈C\theta_1\mathbf{x}_1+\theta_2\mathbf{x}_2\in Cθ1​x1​+θ2​x2​∈C.

The conic hull of a set CCC is the set of all conic combinations of points in CCC.

{θ1x1+…+θkxk∣xi∈C,θi≥0,i=1,…,k}\left\{\theta_{1} \mathbf{x}_{1}+\ldots+\theta_{k} \mathbf{x}_{k} \mid \mathbf{x}_{i} \in C, \theta_{i} \geq 0, i=1, \ldots, k\right\} {θ1​x1​+…+θk​xk​∣xi​∈C,θi​≥0,i=1,…,k}

  • Dual cone

    Let KKK be a cone, its dual cone K∗K^*K∗ satisfies:

    K∗={y∣x⊤y≥0,∀x∈K}K^* = \{\mathbf{y\mid \mathbf{x}^\top \mathbf{y}\ge 0, \forall\mathbf{x}\in K}\} K∗={y∣x⊤y≥0,∀x∈K}

    K∗K^*K∗ is always closed convex. K1⊆K2K_1\subseteq K_2K1​⊆K2​ implies K2∗⊆K1∗K_2^*\subseteq K_1^*K2∗​⊆K1∗​. K∗∗K^{**}K∗∗ is the closure of the convex hull of KKK.

    Examples:

Operators that preserve convexity#

Intersection#

SαS_\alphaSα​ is convex ∀α∈A⇒⋂α∈ASα\forall \alpha\in\mathcal{A}\Rightarrow\bigcap_{\alpha\in\mathcal{A}}S_\alpha∀α∈A⇒⋂α∈A​Sα​ is convex (can be intersection of an infinite number of sets)

Affine function#

S⊆RnS\subseteq \R^nS⊆Rn is convex, f(x)=Ax+b:Rn→Rm⇒f(S)={f(x)∣x∈S}f(\mathbf{x})= \mathbf{Ax}+\mathbf{b}:\R^n\to \R^m\Rightarrow f(S) =\{f(\mathbf{x})\mid \mathbf{x}\in S\}f(x)=Ax+b:Rn→Rm⇒f(S)={f(x)∣x∈S} is convex

Similarly, the inverse image of SSS under fff: f−1(S)={x∣f(x)∈S}f^{-1}(S)=\{\mathbf{x}\mid f(\mathbf{x})\in S\}f−1(S)={x∣f(x)∈S} is also convex.
